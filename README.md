# bank-management-system

This is a python-based project that simulates basic banking operations using MySQL as the backend database.

## features
- creates a new bank account
- views account details
- updates phone number or email
- deletes an account
- menu-driven interface


## technologies-used
- python
- MySQL
- mysql-connector-python

## project structure
- bank.py: main application file
- mysql database: stores account information
